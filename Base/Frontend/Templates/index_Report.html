Webbasierter Störungserfasser Für IT-Störunger

    Bitte tragen Sie im Formular die folgenden Informationen ein:

    Titel der Störung: Geben Sie hier eine kurze Überschrift der Störung ein (z.B. „Drucker funktioniert nicht“).

    Beschreibung: Beschreiben Sie die Störung möglichst genau (z.B. „Der Drucker im Büro 3A reagiert nicht, obwohl er eingeschaltet ist“).

    Name des Meldenden: Tragen Sie hier Ihren Namen ein.

    Kategorie: Wählen Sie die passende Kategorie aus (Drucker, Software, Netzwerk, Sonstiges).

    Nachdem Sie alle Felder ausgefüllt haben, klicken Sie auf den Senden-Button.

    Das System fragt Sie dann, ob alle Angaben korrekt sind.

    Wenn Sie bestätigen, wird die Meldung in die untenstehende Tabelle eingetragen.

    Am Ende sehen Sie eine Übersichtstabelle mit früheren Störungsmeldungen.

    Bitte prüfen Sie vor dem Absenden, ob Ihr Problem bereits gemeldet wurde, um doppelte Einträge zu vermeiden.
